# Python Crud (Flask - React)

---

## Running the project

    Create .env file inside src folder and fill it with the following variables:
      MONGO_URI = "mongodb://..."

    cd backend

    source venv/Scripts/activate

    python /src/app.py

    Visit http://localhost:5000/users to see the users list.
